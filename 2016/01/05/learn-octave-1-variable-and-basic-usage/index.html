Octave 学习记录之基础用法

  • Octave
  • 学习记录

简介:

Octave是一个GNU开源的免费的高等直译语言. 其主要用作数值计算(numerical computation). 其与MATLAB相似, 习惯matlab的童鞋可以轻松上手.
Octave可以解决线性,非线性的众多数学问题, 同时Octave还内置了画图工具, 可以方便的将得到的结果用图像的方式表现出来, 现在的最新版本是在2015年5月发布的Octave 4.0.0. Octave可以应用在机器学习(machine learning)等方面.

安装

安装可以从官网下载二进制的安装包也可以通过软件管理工具下载, 像mac用户可以通过brew和macport下载. 具体细节请参照官网介绍.

在mac下建议使用更加稳定的octave-cli命令行版本

矩阵赋值与初始化

矩阵变量赋值与普通数值变量相同. 矩阵由[ ]初始化, 同行数字由空格或逗号( , )分隔, 不同行之间由分号( ; )分隔

通过[start:step:end]可以生成一个从start起, 步长为step, 终止于end的向量(一维矩阵)

逆矩阵

逆矩阵可以由函数pinv(A)得到, 其中p是pseudo的意思. 所以如果矩阵A不可逆仍可以得到一个逆矩阵, 但矩阵A与其逆矩阵之积不为单位矩阵.

元素级运算

Octave同样可以进行两个矩阵元素级别的乘除运算, 只需在运算符前加上一个 . 即可.

常用内置函数

除了之前提到的那些常用内置函数以外还有以下函数常用到

size(A) / length(A) 返回矩阵的大小

返回值依旧为一个矩阵

length(A)返回行数与列数中的的最大值, 常用于向量

flipud(A) 翻转给定矩阵
max(A) 返回向量的中的最大值或二维矩阵每个向量的最大值

这里可以过两次取最大值得到二维矩阵的最大元素

min(A) 返回向量的中的最小值或二维矩阵每个向量的最小值

同最大值 max(A) 用法相同

format(opt) / format opt 改变数值有效数字显示位数 默认为short

format函数可设置数值显示的有效数字位数, 一共有两个值, long 长显现, short 短显示, 默认为short
