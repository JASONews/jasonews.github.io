if - 条件语句

基本格式

其中 if 条件下的代码必须要由大括号 {} 包裹

if 中的 local variable

在if语句的开始可以像C中的for语句那样,初始化一个本地变量. 这个变量的生命周期包含:

  • 初始化时的if block
  • 同一个if下的其他block 如 else, else if 等

变量初始化的方式只能是 v := val implicit initialization的方式

switch

switch语句与C类的语言有些的变化.

基本格式

switch语句有以下几点需要注意

  • 在语句开始可以像之前的if 和 for 语句一样初始化一个本地变量,而其语法要求也一样
  • 在golang中的每个case都是自动break的, 也就是说它并不具备C中的fallthrough的特性. 但是, 也可以通过添加fallthrough 来达到这样的特性.

采用这种无判断语句的switch可以让多if语句看起来更加清楚
