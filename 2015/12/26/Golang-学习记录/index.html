Golang 学习记录之变量(Variable)

  • golang
  • 学习记录

Variable 共有8种basic types

  1. int

    同时也可以使用给定长度的signed int值: int8 int16 int32 int64

  2. uint

    同int一样可以使用给定长度的值 uint8 uint16 uint32 uintt64 uintptr

    值得一提的是, uintptr 是pointer(指针) 储存地址. int, uint, uintptr 会随着不同位数的机器采用不同的长度

  3. rune

    用来表示Unicode
    是uint32的alias(又名)

  1. byte

    是uint8 的 alias

  2. float32 float64

    float 有两种长度的type: 32bits or 64bits 类似java中的float与double

  3. complex64 complex128

    同 int 与 float 一样 需要指定所需长度 64 or 128 bits

    用来表示复数 如: 1+2i

  4. string

    字符串变量

  5. bool

    boolean 变量
